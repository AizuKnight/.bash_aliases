# How to Use
* Append the following bash script into `~/.bashrc` if there is not.
  ```bash
  if [ -f ~/.bash_aliases ]; then
    . ~/.bash_aliases
  fi
  ```
* Copy your favorite file into `~/` and change its name to `.bash_aliases`.
* Run the following command.
  ```bash
  source ~/.bash_aliases
  ```
That's it. Thank you.
